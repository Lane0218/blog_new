## 实验内容
1. TCP 连接的三次握手、四次挥手分析：详细分析 TCP 在建立连接时所采用的三次握手过程，以及释放连接时所采用的四次挥手过程，包括每个阶段的报文交换和状态转换。
2. TCP 拥塞控制：学习如何使用时间序列图来绘制 TCP 数据传输过程中的拥塞控制机制，以便更好地理解数据流动和网络拥塞情况。
